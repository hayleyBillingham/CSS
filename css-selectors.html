CSS Selectors

  • A CSS Selectors selects the HTML element(s) you want to style
  • CSS Selectors are used to find (or select) the HTML elements you want to style
  • You can divide CSS selectors into five categories
    • Simple selectors: selects elements based on name, id, class
    • Combinator selectors: selects elements based on specific relationship between them
    • Pseudo-class selectors: selects elements based on a certain state
    • Attribute selectors: selects elements based on an attribute or attribute value

The CSS element selector

  • In this example all <p>elements will be center-aligned, with a red text colour:
    p {
      color: red;
      text-align: center;
    }

The CSS ID Selector

  • The ID selector uses the id attribute of an HTML element to select a specific element
  • The ID of the element is unique within a page, so the ID selector is used to select on unique element
  • To select an element with a specific ID, write a hash (#) character, followed by the id of the element
  • Note an ID cannot start with a number
  • #para1 {
      text-align: center;
      color: red;
    }
