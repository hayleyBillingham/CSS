CSS Overview

  • CSS is the language we use to style an HTML document.
  • CSS describes how HTML elements should be displayed.
  • CSS Stands for Cascading Style Sheets
  • CSS saves a lot of work. It can control the layour of multiple web pages all at once
  • CSS exteranl style shees are stored in CSS files
  • HTML was never intended to contain tags for formatting a web page, to add colour and fonts the world wide web consortium (W3C) created CSS
  • A CSS rule consists of a selector and a declaration block.
    • The selector points to the HTML element you want to style
    • The declaration block contains one or more declartions seperated by semicolons
    • Each declaration includes a CSS property name and a value, seperated by a colon.
    • Multiple CSS declartions are seperated with semicolons, and declarion blocks are surrounded by curly braces
  • In this example all <p>elements will be center-aligned, with a red text colour:
    p {
      color: red;
      text-align: center;
    }
  • Above example explained
    • p is a selector in CSS (it points to the HTML element you want to style: <p>).
    • color is a property, and red is the property value
    • text-align is a property, and center is the property value
